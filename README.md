# Mini-projet Algorithmique et Programmation — Ivrognes et Dendrites

**Auteurs :** Germain Caudal Lhermite et Antoine VENTRE  
**IFIE1 – Octobre 2025 – IMT Mines Albi**

## Description
Ce projet simule des balades aléatoires (“balade de l’ivrogne”) pour produire des images :  
1. Trois ivrognes en goguette : chaque ivrogne laisse une trace colorée sur une image.  
2. Dendrites et ivrognes : formation de structures ressemblant à des dendrites ou cristaux à partir de points noirs et d’ivrognes errants.
